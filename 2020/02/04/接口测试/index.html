接口测试体系

  • 传统的狭义的接口测试。http、rpc、tcp协议的接口测试
  • 泛接口体系。除了狭义接口测试之外，还包括sdk的测试、so、dll文件的api接口测试、数据库、消息管道的测试，各种协议测试

协议分析工具

  • 网络嗅探：TcpDump ，wireshark
  • Proxy
    • 知名： fiddler（windows） anyproxy（全平台） mitmproxy
    • 推荐： Charles（全平台）， burpsuite（全平台java）
  • 分析工具与协议客户端工具
    • curl
    • postman
      chrome DevTool
  • 从上到下分析协议
    chrome的networks
    curl http协议的发送工具 -vv更清晰的理解http协议

httprunner

使用hrun –starparoject name 生成httprunner项目。使用charles 导出请求为har格式，运行命令 har2case name.har -2y 生成yml文件（不加-2y为json文件）运行hrun name.yml 执行测试，生成测试报告

  • poetry安装
    Poetry是啥？
    是一个Python虚拟环境和依赖管理工具，另外它还提供了包管理功能，比如打包和发布。
    可以用来管理python库和python程序。

  • travis

本地新建.travis.yml文件


使用github账号登录travis网站后，添加该项目，以后每次更新仓库，都会自动执行文件了,还可在redame中添加build status
