stf

安装

brew install rethinkdb graphicsmagick zeromq protobuf yasm pkg-config
npm install -g stf （安装过程中可能应为node版本不对报错，可以执行npm install -g n，sudo n 8.14.0）

启动

新建stf 文件夹，该目录下 执行 nohup rethinkdb & ,nohup stf local --public-ip <本地ip>
